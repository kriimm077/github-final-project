# Simple Interest Calculator

## Description
This project is a simple calculator that computes **Simple Interest** using the standard formula:

**Simple Interest = (Principal × Rate × Time) / 100**

The calculator accepts the principal amount, rate of interest, and time period as inputs and returns the calculated simple interest.

## Formula
SI = (P × R × T) / 100

Where:
- **P** = Principal Amount
- **R** = Rate of Interest (per year)
- **T** = Time (in years)

## Features
- Calculates simple interest quickly and accurately.
- Easy-to-understand implementation.
- Beginner-friendly project for learning Git and GitHub.

## Example

Input:
- Principal = 1000
- Rate = 5%
- Time = 2 years

Output:
- Simple Interest = 100
